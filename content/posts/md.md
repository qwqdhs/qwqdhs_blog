1. 标题层级（H1-H6）
markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
注意：# 与文字之间必须有空格。

2. 文字修饰（强调、删除、高亮）
markdown
*斜体*         或  _斜体_
**粗体**       或  __粗体__
***粗斜体***   或  ___粗斜体___
~~删除线~~      （划线）
==高亮标记==    （需要平台支持，如 Obsidian/Notion）
`行内代码`      （反引号包裹）
关于下划线与颜色：原生 Markdown 不支持。必须内嵌 HTML 标签：

html
<u>下划线文字</u>
<span style="color:red">红色文字</span>
<span style="background:yellow">背景色</span>
<font size=4>字体大小</font>
3. 换行与段落
换行：在行尾加 两个空格 再回车。

新段落：直接空一行。

强制换行：使用 <br> 标签。

4. 列表（有序、无序、任务）
markdown
# 无序列表（- 或 * 或 + 均可）
- 项目 1
  - 子项目（缩进 2 或 4 个空格）
* 项目 2

# 有序列表（数字加英文句号）
1. 第一项
2. 第二项
   1. 子项 1（缩进）
   
# 任务列表（GFM 扩展）
- [ ] 未完成
- [x] 已完成
5. 链接与图片
markdown
# 行内链接
[显示文字](https://example.com "鼠标悬停标题")

# 自动链接（直接展示网址）
<https://example.com>

# 引用式链接（便于复用）
[显示文字][引用ID]
[引用ID]: https://example.com

# 图片（感叹号+方括号+圆括号）
![替代文字](图片链接.jpg "标题")

# 带链接的图片（点击图片跳转）
[![替代文字](图片链接.jpg)](https://example.com)
6. 代码块（围栏与缩进）
markdown
# 围栏式（推荐，可指定语言实现高亮）
```python
print("Hello World")
缩进式（每行前加 4 个空格或 1 个 Tab）
print("这属于代码块")

text

### 7. 引用块（Blockquotes）
```markdown
> 这是一级引用。
>> 这是二级嵌套引用（使用多个 >）。
> 回到一级。
>
> 引用内也可以放列表、标题等。

# GFM 专用：提醒框（Alert）
> [!NOTE] 提示
> 这是个普通提示。
> [!WARNING] 警告
> 这是个警告框。
8. 表格（GFM 扩展）
markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容1  | 内容2    | 内容3  |
| 内容4  | 内容5    | 内容6  |
语法要点：--- 代表分隔线；冒号 : 控制对齐方向。

9. 分隔线（水平线）
单独一行，使用 3 个以上的 -、* 或 _，中间可加空格：

markdown
---
***
___
10. 脚注（Footnote）
markdown
这里需要备注一个信息。[^1]

[^1]: 这是脚注的具体内容。
11. 数学公式（LaTeX）
需要平台安装 MathJax/KaTeX 插件（如 Typora、Obsidian、GitHub）。

markdown
# 行内公式：$ 包围
$E = mc^2$

# 独立公式块：$$ 包围
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
12. 图表（Mermaid 流程图/时序图）
GitHub、GitLab、Typora 等支持。

graph TD
    A[开始] --> B{判断条件}
    B -- 是 --> C[执行]
    B -- 否 --> D[结束]
13. 转义字符（显示特殊符号）
如果你想显示 * 或 # 本身而不触发格式，前面加反斜杠 \：

markdown
\* 这样就不会变成斜体了
\# 这样就不会变成标题了
可转义字符包括：\   * _ {} [] () # + - . ! |。
